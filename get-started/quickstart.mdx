### Making Your First API Request

<AccordionGroup>

<Accordion icon="message-bot" title="HTTPie Example">
  ```bash
  http POST https://dev.undrstnd-labs.com/api \
  x-api-key:YOUR_API_KEY_UNDRSTND \
  stream=false \
  modelId="MODEL_ID" \
  messages:='[{"name": "system", "content": "Hello, how can I assist?", "role": "system"}, {"name": "user", "content": "What is the weather in San Francisco?", "role": "user"}]'
  ```
</Accordion>
<Accordion icon="rocket" title="cURL Example">
    ```bash
    curl -X POST https://dev.undrstnd-labs.com/api \
      -H "x-api-key: YOUR_API_KEY_UNDRSTND" \
      -H "Content-Type: application/json" \
      -d '{
        "stream": false,
        "modelId": "MODEL_ID",
        "messages": [
          {
            "name": "system",
            "content": "Hello, how can I help you?",
            "role": "system"
          },
          {
            "name": "user",
            "content": "What is the weather like in San Francisco?",
            "role": "user"
          }
        ]
      }'
  ```
</Accordion>
<Accordion title="Postman Example" icon="postman">
  1. Open Postman and create a new POST request.
  2. Enter the API endpoint `https://dev.undrstnd-labs.com/api`.
  3. In the Headers tab, add:
      - `x-api-key` with your API key.
      - `Content-Type` with `application/json`.
  4. In the Body tab, choose raw and select JSON format. Add this body:
  ```json
    {
    "stream": false,
    "modelId": "MODEL_ID",
    "messages": [
      {
        "name": "system",
        "content": "Hello, how can I help you?",
        "role": "system"
      },
      {
        "name": "user",
        "content": "What is the weather like in San Francisco?",
        "role": "user"
      }
    ]
  }
  ```
  5. Click Send to make the request.
</Accordion>

</AccordionGroup>

## Check Model Status

To view available models or check the status of an individual model, use the following endpoints:

<CardGroup>

<Card title="List All Models" icon="paintbrush" href="https://dev.undrstnd-labs.com/api/models/info">
  Get a list of all available models and their details.
</Card>

<Card
  title="Get Specific Model Info"
  icon="square-code"
  href="GET https://dev.undrstnd-labs.com/api/models/{MODEL_ID}/info"
>
  Retrieve detailed information about a specific model.
  Replace {MODEL_ID} with the actual model ID to get detailed information about a specific model.
</Card>

</CardGroup>
